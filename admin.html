Panel de Administración - Propiedades Collipulli

Carga propiedades con fotos, videos y descripción para generar el JSON listo para publicar.

Flujo de trabajo

1. Completa los datos de la propiedad.
2. Selecciona fotos y/o videos.
3. Sube a S3.
4. Copia el JSON generado.
5. Pégalo dentro de propiedades.json.

Puedes escribirlo ya formateado. Luego automatizaremos este campo.

Esta descripción se usará en la ficha individual de la propiedad.

• Puedes seleccionar múltiples archivos a la vez.

• La primera imagen encontrada se usará como portada (imagenUrl).

• Más adelante el sistema generará también galeria y videos.
